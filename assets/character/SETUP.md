# Adding a 3D character to Galatea

Place your converted GLB files here:

```
assets/character/
├── idle.glb    ← character mesh + idle animation  (required)
├── talk.glb    ← talking animation                (optional but recommended)
└── think.glb   ← thinking animation               (optional)
```

See the main README for the full Mixamo → Blender → GLB setup guide.
